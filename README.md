<h1>BookMyHotel</h1>
<p>Hotel booking website developed using ViteJs.</p>

<h1>Tools Used</h1>
<ul>
  <li>ViteJs</li>
  <li>React Lazy Loading</li>
  <li>ChakraUI</li>
  <li>JWT Web Token</li>
  <li>Bcrypt Hashing</li>
  <li>NodeJs</li>
  <li>ExpressJs</li>
  <li>MongoDB</li>
  <li>Multer</li>
  <li>React Routing</li>
  <li>User Context</li>
  <li>Axios</li>
</ul>
